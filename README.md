# Premium Personal Portfolio

A modern, high-performance personal portfolio website built with **Next.js 15**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**. Featuring a dark editorial aesthetic with refined motion design, smooth animations, and a fully responsive layout.

## ✨ Features

### Design & UX
- **Dark Editorial Aesthetic** - Clean, minimal, cinematic, and confident visual design
- **Premium Typography** - Bold headlines with strong hierarchy and modern sans-serif body text
- **Refined Motion Design** - Subtle, intentional animations that enhance rather than distract
- **Fully Responsive** - Elegant scaling across mobile, tablet, and desktop devices

### Technical Excellence
- **Next.js 15 App Router** - Latest Next.js with modern React features
- **TypeScript** - Full type safety for robust code
- **Tailwind CSS** - Utility-first styling with custom configuration
- **Framer Motion** - Production-grade animations with performance optimization

## 🚀 Getting Started

### Installation

```bash
cd portfolio
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000)

## 📁 Project Structure

```
src/
├── app/
│   ├── layout.tsx          # Root layout
│   ├── page.tsx            # Main page
│   └── globals.css         # Global styles
├── components/             # Reusable components
├── hooks/                  # Custom hooks
└── lib/                    # Utilities
```
